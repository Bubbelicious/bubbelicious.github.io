One of the functions that I worked on were making wireframes for the screens that are used when the player choses a Play action. The actions in question are: Join Game or Create Game.
The designs had to include a screen for joining a specific room, loading screens, and failed screens.

Join Game

Screen to choose which game room the player want to join.

Loading screen when player has chosen the game room they want o join.

Failed screen for when the chosen game room could not load for some reason.

Create Game

Loading screen when player has their own game room for others to join.

Failed screen for when the created game room could not load for some reason.
